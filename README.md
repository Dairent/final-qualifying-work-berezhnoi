## 📖 О проекте

Этот проект представляет собой полностью функциональный прототип игры в жанре **пошаговой стратегии (Turn-Based Strategy)** с элементами рогалика. Разработан в рамках выпускной квалификационной работы бакалавра по направлению «Прикладная информатика» в Южном федеральном университете.

Цель работы — реализовать ключевые механики жанра TBS, создать уникальный игровой опыт с процедурной генерацией карт, глубокой тактической системой и высокой реиграбельностью.

---

## ✨ Особенности

- **Выбор класса** — на старте доступны три героя: **Воин**, **Маг** и **Лучник**. У каждого уникальные навыки, характеристики и стиль игры.
- **Процедурная генерация карт** — каждое сражение происходит на уникальном ландшафте, создаваемом с помощью шума Перлина и клеточного автомата.
- **Пошаговая боевая система** — классическая тактика с очередностью ходов, очками движения, зонами атаки и расчётом пути (алгоритм Дейкстры).
- **Пять типов противников** — каждый враг обладает уникальным ИИ-поведением (ближний бой, дальний бой, оборонительная тактика).
- **Статус-эффекты** — «Горение», «Мокрый» и «Скрыт» влияют на тактику и взаимодействие с окружением.
- **Карта пути (надземный слой)** — после боя игрок выбирает одну из трёх дорог: **бой**, **костёр** (восстановление) или **магазин** (покупка предметов).
- **Система сохранений** — прогресс сохраняется в JSON-файлы, поддерживаются несколько слотов.
- **Адаптивный интерфейс** — подсветка доступных действий, подсказки при наведении, настраиваемые параметры звука и экрана.

## 🎮 Геймплей

1. В главном меню выберите **«Новая игра»**.
2. На экране выбора героя ознакомьтесь с характеристиками и выберите класс.
3. Сражение начинается на процедурно сгенерированной карте.
   - Используйте **очки движения** для перемещения.
   - Наносите урон базовыми атаками и специальными навыками.
   - Учитывайте статус-эффекты и ландшафт (вода, деревья, укрытия).
4. После победы над всеми врагами выберите путь на карте:
   - ⚔️ **Бой** — новое случайное сражение с повышенной сложностью.
   - 🔥 **Костёр** — полное восстановление здоровья и зарядов умений.
   - 🏪 **Магазин** — покупка случайных предметов за монеты опыта.
5. Финальный босс адаптируется под вашу стратегию — каждый забег уникален!

## 🛠️ Технологии

- **Движок** — Godot Engine 3.6
- **Язык сценариев** — GDScript (основной), C# (опционально)
- **Графика** — 2D-тайловые карты, анимации спрайтов
- **Алгоритмы** — шум Перлина, клеточный автомат, алгоритм Дейкстры, A* (для ИИ)
- **Сохранения** — JSON-сериализация с защитой от повреждений
- **Инструменты** — встроенный редактор сцен, система сигналов, TileMap

## 🚀 Установка и запуск

### Требования
- Godot Engine 3.6 (или выше)
- Операционная система: Windows, Linux или macOS

### Инструкция
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш-username/tbs-roguelike-prototype.git
Откройте проект в Godot:

Запустите Godot → нажмите Импорт → выберите папку с проектом.

Нажмите Запустить (F5) или экспортируйте игру под нужную платформу.

Примечание: Проект не требует дополнительных библиотек или настроек — всё работает «из коробки».

🖥️ Управление
Действие	Клавиша / Мышь
Выбор действия (атака, навык, шаг)	Нажатие на кнопки в нижней панели
Перемещение / атака	Клик по подсвеченной клетке
Подсказка	Наведение мыши на юнита или объект
Завершить ход	Кнопка «Конец хода» (справа внизу)
