## 📖 О проекте

Этот проект — полностью функциональный прототип игры в жанре **пошаговой стратегии (Turn-Based Strategy)** с элементами рогалика.

Разработан в рамках выпускной квалификационной работы бакалавра по направлению **«Прикладная информатика»** в Южном федеральном университете на тему:
> *«Реализация ключевых механик жанра TBS в рамках игрового проекта»*

**Цель работы** — реализовать ключевые механики жанра TBS, создать уникальный игровой опыт с процедурной генерацией карт, глубокой тактической системой и высокой реиграбельностью.

---

## ✨ Особенности

| | |
|---|---|
| 🧙 **Выбор класса** | Три героя на старте — **Воин**, **Маг** и **Лучник**, каждый с уникальными навыками, характеристиками и стилем игры |
| 🗺️ **Процедурная генерация карт** | Каждое сражение проходит на уникальном ландшафте, созданном с помощью шума Перлина и клеточного автомата |
| ⚙️ **Пошаговая боевая система** | Классическая тактика: очерёдность ходов, очки движения, зоны атаки, расчёт пути по алгоритму Дейкстры |
| 👹 **Пять типов противников** | У каждого врага своё ИИ-поведение — ближний бой, дальний бой, оборонительная тактика |
| 🔥 **Статус-эффекты** | «Горение», «Мокрый» и «Скрыт» влияют на тактику и взаимодействие с окружением |
| 🛤️ **Карта пути** | После боя — выбор между **боем**, **костром** (восстановление) и **магазином** (покупка предметов) |
| 💾 **Система сохранений** | Прогресс сохраняется в JSON, поддерживаются несколько слотов |
| 🖱️ **Адаптивный интерфейс** | Подсветка доступных действий, подсказки при наведении, настройки звука и экрана |

## 🎮 Геймплей

1. В главном меню выберите **«Новая игра»**.
2. На экране выбора героя изучите характеристики и выберите класс.
3. Сражение начинается на процедурно сгенерированной карте:
   - используйте **очки движения** для перемещения;
   - наносите урон базовыми атаками и специальными навыками;
   - учитывайте статус-эффекты и ландшафт (вода, деревья, укрытия).
4. После победы над всеми врагами выберите путь на карте:

   | Путь | Эффект |
   |---|---|
   | ⚔️ **Бой** | Новое случайное сражение с повышенной сложностью |
   | 🔥 **Костёр** | Полное восстановление здоровья и зарядов умений |
   | 🏪 **Магазин** | Покупка случайных предметов за монеты опыта |

5. Финальный босс адаптируется под вашу стратегию — **каждый забег уникален**!

## 🛠️ Технологии

- **Движок** — Godot Engine 3.6
- **Язык сценариев** — GDScript
- **Графика** — 2D-тайловые карты, анимации спрайтов
- **Алгоритмы** — шум Перлина, клеточный автомат, алгоритм Дейкстры, A* (для ИИ)
- **Сохранения** — JSON-сериализация с защитой от повреждений
- **Инструменты** — встроенный редактор сцен, система сигналов, TileMap

## 🚀 Установка и запуск

### Требования
- Godot Engine **3.6**
- ОС: **Windows**

### Инструкция

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ваш-username/tbs-roguelike-prototype.git
```

2. Откройте проект в Godot:
   - запустите Godot → **Импорт** → выберите папку с проектом;
3. Нажмите **Запустить** (`F5`) или экспортируйте игру под нужную платформу.

> 💡 Проект не требует дополнительных библиотек или настроек — всё работает «из коробки».

## 🖥️ Управление

| Действие | Клавиша / Мышь |
|---|---|
| Выбор действия (атака, навык, шаг) | Нажатие на кнопки в нижней панели |
| Перемещение / атака | Клик по подсвеченной клетке |
| Подсказка | Наведение мыши на юнита или объект |
| Завершить ход | Кнопка «Конец хода» (справа внизу) |
